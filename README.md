# 🖥️ AbhiDesk — Personal Command Centre

A self-contained browser dashboard built as a single folder of HTML files.  
No server. No login. No dependencies. Just open and use.

**Live:** [abhidesk.github.io/AbhiDesk](https://[YOUR_USERNAME].github.io/AbhiDesk/)

---

## 📑 Panels

| Tab | What it does |
|-----|-------------|
| 📅 DayFlow | Visual block planner — drag time blocks onto a 30-day timeline |
| 🫁 Breathe | Breathing exercise timer with presets (4-7-8, Box, Wim Hof etc.) |
| 💻 Code Tracker | Coding session logger with task queue and challenge editor |
| 📊 2026 Daily Log | Daily wellness tracker — wake time, gym, meditation, ratings |
| 📋 Checklist Sheet | Embeds a live Google Sheet for on-the-go notes |

---

## ✨ Features

- ☀️ / 🌙 Dark and light mode toggle — syncs across all panels
- Tabs are renameable (double-click) and drag-to-reorder
- 🚀 **Productivity Started** button stamps current time as work start
- Gap between wake-up and work start auto-calculates
- Weekly and yearly stats in sidebar
- All data saved locally in your browser (localStorage) — fully private
- Export 2026 log to CSV anytime

---

## 🚀 How to use

1. Download all 4 files into one folder
2. Open `index.html` in any browser
3. Everything works offline — no internet needed (except Google Fonts)
